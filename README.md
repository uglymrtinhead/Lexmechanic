## `// OVERVIEW`

**Lexmechanic** is an AAR tracking plugin for the Immortal IXth Legion Blood Angels — built for the Logisticum, deployed to the Chapter.

It automatically scans the After-Action Report channel, calculates AAR points per the Chapter's regulations, and surfaces evidence links for every counted entry. No bot. No server. No cost. No downtime.

It runs entirely on the user's machine through BetterDiscord — making no automated actions on behalf of the user, which keeps it within Discord's Terms of Service.

---

## `// CURRENT FEATURES`

### `◆ SQUAD TRACKING`
Create a squad and add the Brothers you want to monitor. The plugin narrows all scans and reports to the selected roster — filter by squad, view the whole Chapter, or anything in between.

### `◆ DATE RANGE FILTERING`
Set a `FROM / TO` date range to drill into any day, week, or month. Run a weekly report. Audit a specific operation window. The underlying lifetime data is always preserved — filters are display-only.

### `◆ EVIDENCE TRACKING`
Click any Brother's name in the leaderboard. A drill-down opens showing every AAR that was counted — date, difficulty, points, and a direct **`VIEW →`** link to the original Discord message.

> That way, we're not trusting a machine to do the counting for us. The actual AARs used to factor a Brother's points are easily available for viewing in-browser. 

### `◆ MULTI-TAB LEADERBOARD`
View totals broken down by **TOTAL / OPS / SIEGE / PVP** — with optional detail mode showing the full breakdown per Brother at a glance.

## `// COMING FEATURE`

### `◆ WEEKLY REPORT GENERATION`
Automated formatted report output for Sergeant submissions — one click, paste into the reporting channel, done.

## `// INSTALLATION`

```
1. Download and install BetterDiscord
   → betterdiscord.app

2. Place the plugin file in your plugins folder
   → %APPDATA%\Roaming\BetterDiscord\plugins\Lexmechanic.plugin.js

3. Open Discord → Settings cogwheel (bottom left)

4. Scroll down to Plugins under the BetterDiscord section

5. Activate Lexmechanic
```

A **`LEX`** button will appear in the Discord toolbar. Click it to open the ledger.

---

## `// TROUBLESHOOTING`

| Symptom | Fix |
|---|---|
| Plugin not responding | Toggle off and on in Plugin settings |
| Data not appearing after scan | Disable, re-enable plugin, run scan again |
| Persistent errors | Delete plugin + config from plugins folder, reinstall |

Config file location: `%APPDATA%\Roaming\BetterDiscord\plugins\Lexmechanic.config.json`

---

## `// COMPLIANCE`

> **Following standardized AAR reporting procedures is HIGHLY recommended.**

The plugin parses AAR posts based on structure. Non-standard formatting — incorrect field labels, missing Members section, unlisted difficulty tags — may result in posts being skipped. The Evidence Tracking feature exists so that discrepancies can be caught and corrected.
